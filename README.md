# demoblaze_Manual_assignment
This repository contains a Manual Testing assignment for functional testing of an E-commerce web application: Demoblaze (https://www.demoblaze.com/).
The objective of this project is to design, execute, and document comprehensive manual test cases covering critical application functionalities such as user authentication, product browsing, cart operations, checkout and logout. 
Both positive and negative test scenarios are included
